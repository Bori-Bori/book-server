# book-server
boribori book-server

# How to Run?

1. First Step

```shell
$ git clone https://github.com/Bori-Bori/book-server
```

2. Second Step

locate directory

3. Third Step

```shell
$ ./gradlew build
```

4. Fourth Step

```shell
$ nohup java -jar ~.jar & /dev/null
```
